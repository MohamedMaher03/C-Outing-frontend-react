# Common Components

Reusable, generic components used across the application.

## Planned Components

- **Button** - Primary, secondary, tertiary variants
- **Card** - Container for content with shadow and border radius
- **Modal** - Dialog component with overlay
- **Spinner** - Loading indicator
- **Toast** - Notification messages
- **Header** - Navigation header with language switcher
- **Footer** - Application footer
- **Input** - Text input with validation
- **Select** - Dropdown select component
- **Checkbox** - Checkbox input component
- **Radio** - Radio button component
- **Badge** - Small label/tag component
- **Avatar** - User avatar with initials or image
- **Stars** - Star rating display
- **Pagination** - Page navigation
- **SearchBar** - Search input with suggestions
- **Empty State** - No data state UI
