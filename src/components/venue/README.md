# Venue Components

Components for displaying venue details, reviews, and interactions.

## Planned Components

- **VenueDetailView** - Full venue details page
- **VenueCard** - Compact venue card (used in lists)
- **VenueGallery** - Image carousel for venue photos
- **ReviewList** - List of venue reviews with sentiment
- **ReviewForm** - Form to submit venue rating and comment
- **VenueMap** - Map view with venue location (Google Maps integration)
- **VenueMeta** - Metadata display (rating, sentiment, category, hours)
- **FavoriteButton** - Button to add/remove venue from favorites
