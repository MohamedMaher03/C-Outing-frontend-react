# Context Input Components

Components for user context and intent input.

## Planned Components

- **MoodSelector** - Select mood (relaxed, energetic, social, romantic)
- **CompanionSelector** - Select companion (solo, friends, family, partner)
- **BudgetSelector** - Select budget level (low, medium, high)
- **TimeSelector** - Select time of day (morning, afternoon, evening, night)
- **LocationInput** - Location input with geolocation option
- **WeatherDisplay** - Current weather based on location
- **ContextPanel** - Combined context input panel
