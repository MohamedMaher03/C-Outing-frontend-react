# Recommendation Components

Components for displaying and interacting with venue recommendations.

## Planned Components

- **RecommendationsList** - Infinite scroll list of recommendations
- **RecommendationCard** - Individual recommendation card
- **ContextInput** - Context/intent selection widget (mood, companion, budget, time)
- **RecommendationFilters** - Advanced filtering options
- **EmptyRecommendations** - Empty state when no recommendations
